== Description ==

### Preloader 

This plugin will enable custom background color and custom preloader image url in your wordpress site.

= Demo =

Go to [Plugin Demo Site](http://sumonhasan.com/plugins/quick-preloader/)


### The Features

* Admin Options.
* Easy to use.
* Custom  Image.
* Custom  Background Color
* jQuery & CSS3.
* Compatible with Google Chrome, FireFox, Opera, Safari, IE9, IE10, IE11 and with mobile devices like iPad, iPhone, Android, etc.


### More Plugins

* [My Plugins](https://profiles.wordpress.org/infodesignbd#content-favorites)

== Installation ==

### Installation

1. Upload 'the-preloader' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Plugins menu > Quick Preloader
4. Go to Settings > Quick Preloader
